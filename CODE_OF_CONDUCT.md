# Code of Conduct

## Our Standard

This project expects respectful, direct, and technically grounded collaboration.

Examples of positive behavior:

- Sharing reproducible bug reports
- Giving specific, actionable feedback
- Debating ideas without personal attacks
- Respecting contributor time and scope

Examples of unacceptable behavior:

- Harassment or personal attacks
- Bad-faith issue spam
- Publishing private security details in public threads
- Repeatedly pushing scope far beyond the project goals

## Enforcement

Project maintainers may remove comments, close threads, or block participation when conduct damages the collaboration environment.
